# Token Orbit

Token Orbit 的视觉核心是“可计算的能量”：把抽象的额度变化转译为可见的轨道与中心结构。构图采用明确的中心秩序，保证在 macOS 图标从 1024 缩小到 16 像素时依然可识别，整体强调沉稳、精确、长期可用，而不是短期视觉噱头。

色彩策略以深海蓝为底，代表系统稳定性与信任感；以青绿色弧线表达“仍在流动的可用额度”。两者对比强烈但不过饱和，避免在浅色与深色桌面壁纸上出现刺眼溢出。明暗关系以微弱发光和柔和阴影构建空间层次，让图标看起来经过反复校准与精密打磨。

形态语言坚持“中心标识 + 外层轨道”：中心 `T` 作为品牌锚点，轨道弧线作为状态语义。轨道不闭合，保留一个刻意的缺口，暗示额度不是静态总量，而是持续消耗与补给的过程。所有几何形体采用圆角处理，保证科技感与亲和力并存。

工艺要求是高精度与克制。边界、留白、线宽、半径必须像界面组件一样可度量；每一处高光都应服务可读性而非装饰。最终成品应呈现“经过大量迭代后留下的最少元素”，看起来是资深设计师长期推敲后的结果，而不是一次性拼接出来的图形。
